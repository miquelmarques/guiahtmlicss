Els enllaços en HTML

En HTML els enllaços permeten moures més facilment des de una pagina web o moures dese de una pagina web a una d'altra. I un enlaç en HTML pot estar format per una imatge, text o imatge i text.

Per fer un enllaç es fa servir el <a>.

Enllaç amb text

<a href "https://www.google.com"> GOOGLE </a>

Resultat

GOOGLE

Enllaç amb imatge

<a href "https://www.google.com"> <img src="images/Google.png" alt="google.com"> </a>

Resultat

Enllaç amb text i imatge

<a href "https://www.google.com"><img src="images/Google.png" alt="google.com"> GOOGLE </a>

Resultat

GOOGLE

llistes en HTML

Les llistes en HTML poden ser de dos tipus ordenades o no ordenada. La ordenada es la classica llista que segueix un ordre com 1,2,3,4... o a,b,c,d... mentre que les llistes no ordenades son aquelles que tenen un punt, un guiò o el que sigui.

I per fer una llista el que primer has de posar es el següent:
<ul>: llista no ordenada
<ol>: llista ordenada

Llistes ordenades

<ol>
<li>Introducció</li>
<li>Punt 1</li>
<li>Punt 2</li>
<li>Conclusió</li>
</ol>

Resultat

  1. Introducció
  2. Punt 1
  3. Punt 2
  4. Conclusió

Llista no ordenada

<ul>
<li>Introducció</li>
<li>Punt 1</li>
<li>Punt 2</li>
<li>Conclusió</li>
</ul>

Resultat

  • Introducció
  • Punt 1
  • Punt 2
  • Conclusió

Taules en HTML

Les taules presenten informacicó organitzada en files i columnes. Per crear una taula necessitem 4 etiquetes de base:

  • <table>: per crear la taula
  • <tr>: per definir les files
  • <td>: per definir les cel·les
  • <th> : per definir les cel·les que faran de capcalera
  • <td colspan="{valor}">: per poder crear cel·les que ocupin més de una columna
  • <td rowspan="{valor}">: per poder crear cel·les que ocupin més de una fila

Exemple de taula:

<table>
<tr>
<th> TEXT </th>
<th> PREU </th>
<th> DESCOMPTE </th>
</tr>
<tr>
<td> Ous </td>
<td> 12 euros </td>
<td> 20% </td>
</tr>
<tr>
<td rowspan="2"> Pollastre </td>
<td> 17 euros </td>
<td> 30% </td>
</tr>
<tr>
<td colspan="2"> 0 euros (Regal) </td>
</td>
</tr>
</table>

  • Aplicar de forma local
  • Fulls d'estil interns
  • Fulls d'estil externs

Aplicar els estil de forma local

Podem aplicar els estils de CCS directament a un element HTML incloent el atribut style a la declaració de l'element

Regles de l'estil

Les regles per aplicar un estil css estan compostes per dos de principals:

  • Selector: Indica als elements als quals s'aplica l'estil
  • Declaració: es l'apartat on es dona les propietats a un element per posar-li l'estil.

Al final han de tenir més o menys aquesta esritura

Tipus de selectors

Els tipus de selector per fer referncia a un element o varis són els següents:

  • Mitjançant un selector de HTML: En aquest casos es poden fer servir pels elements com <p>, &span> i el td per poder dir que tots els paragrafs els text seleccionat o les casselles tenen aquest estil.
  • Indentificadors: Els elements en HTML es defineixen amb la propietat de (id) que significa que els diferencia de la resta per tant solament es pot fer servir una vegada.
    I en el CSS es fan referencia amb aqueta estructura #id_posat
  • Classes: Son elements en HTMLs que es defineixen amb la propietat (class) que poden tenir-ho més d'un element i que es poden posar a qualsevol element.
    I per fer referencia a ells en el CSS secriu .classe_posat
  • Pseudo classes: Amb les pseudo classes podem definir diferents estils en un mateix element S’aplicarà un estil o un altre en funció dels esdeveniments provocats per l’usuari o de l'estat en el que es trobi l'element.
    I la manera de fer-ho en el CSS és la següent: selector:pseudo-classe
  • Pseudo-elements:S’aplicarà un estil o un altre en funció dels esdeveniments provocats per l’usuari o de l'estat en el que es trobi l'element. En CSS s'escriu de la següent forma: selector::pseudo-element

Per veure els exemplesaquí i el nom de l'arxiu ha obrir es regles_de_estil.hmtl i regles_de_estil.css

Formularis en HTML

En aquest apartat veurem com podem crear formularis en HTML

Un formulari és l’eina utilitzada per recollir la informació introduïda per l’usuari

Creació d'un formulari

El primer que has de fer per crera un formulari es afegir el <form> que és el encarrgeat de que puguem enviar les dades a una base de dades.

On hem de posar una serie de parametres:

  • action: Servei per posar la URL on es guardan les dades.
  • name: Serveix per posar un nom.
  • enctype: Forma en les que es codificaran les dades.
  • method: determina la manera com s'envien les dades (get o post).
  • target: Serveix per determinar on s'ha d'obrir el URL
  • autocomplete: Serveix per activar o desactivar l'opció de autocompletar en els elements de formulari.
  • novalidate: Serveix per permtere enviar dades sense validar

Elements de formularis

En els formularis d'HTML hi ha una serie de tipus de fer entrades de l'usuari:

Elements de formulari

  • input
  • textarea
  • select
  • datalist
  • button
  • fieldset
  • label

Exemple: <input type="text" required pattern="[A-Z]{9,20}-.{2}">

Animacions

Aquesta propietat permet fer animacions en els elements. En que necessitarem una keyframes per poder fer l'animació.

Com fer un keyframe

            @keyframes [nom]{ 

                0%{ 

                    transform:rotate(0deg); 

                } 

                100%{ 

                transform:rotate(360deg); 

                } 

                            } 
 

                            .classe{ 

                            animation:[nom] 2s linear infinite; 

                            }
Exemple d'animació:
Propietat Explicació Exemple Resultat
animation-name posem el nom que hem assignat al element animation-name: [nom]
HOLA GUAPO
animation-duration duracio animació animation-duration:[num]s
animation-timing-function funció de velocitat utilitzada durant l’animació animation-timing-function: linear;
animation-iteration-count nombre de vegades que es repeteix l'animació animation-iteration-count: infinite;
